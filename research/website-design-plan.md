# 龙虾工作台 - 网站设计方案

## 需求分析

### 核心功能
1. **信息展示面板** - 分类展示每日整理的内容
2. **可视化状态** - 像素风格的龙虾工作状态
3. **在线访问** - 部署到服务器，支持远程访问

### 内容板块
- AI 博客日报 (每天 7:00 AM 更新)
- 大咖知识库 (每天 9:00 AM 更新)
- GitHub 项目推荐 (每天 10:00 AM 更新)
- 龙虾实时状态 (心跳同步)

---

## 技术方案

### 方案 A: 静态网站 + 自动部署 (推荐)
```
技术栈:
- 前端: Next.js / Astro (静态生成)
- 样式: Tailwind CSS + 自定义像素风格
- 部署: Vercel / Netlify (免费)
- 数据: Markdown 文件 + Git 仓库
- 更新: GitHub Actions 定时推送
```

**优点**:
- 免费托管
- 自动部署
- 访问速度快
- 维护简单

**缺点**:
- 状态更新有延迟（5-10分钟）
- 不能实时显示龙虾状态

---

### 方案 B: 动态网站 + 服务器
```
技术栈:
- 前端: React + Canvas (像素动画)
- 后端: Node.js + WebSocket
- 部署: 云服务器 (阿里云/腾讯云)
- 数据库: SQLite / PostgreSQL
- 实时: WebSocket 推送状态
```

**优点**:
- 实时状态更新
- 完整的像素游戏效果
- 可交互

**缺点**:
- 需要服务器费用
- 维护复杂

---

### 方案 C: 混合方案 (平衡)
```
技术栈:
- 静态展示: Vercel (内容板块)
- 实时状态: 内网穿透 + 本地服务 (像素办公室)
- 或者: 用第三方服务 (如 Replit) 托管实时部分
```

## 设计风格选项

### 选项 1: 极简像素风
```
配色:
- 背景: #1a1a2e (深蓝黑)
- 强调: #e94560 (珊瑚红)
- 文字: #eaeaea (米白)
- 辅助: #0f3460 (深蓝)

特点:
- 16-bit 像素风格
- 清晰的板块分隔
- 复古游戏感
- 参考: Stardew Valley UI
```

### 选项 2: 赛博朋克像素
```
配色:
- 背景: #0d0221 (深紫黑)
- 强调: #ff3864 (霓虹粉)
- 文字: #00f0ff (青色)
- 辅助: #261447 (深紫)

特点:
- 霓虹光效
- 扫描线效果
- 科技感强
- 参考: Cyberpunk 2077 + 像素
```

### 选项 3: 温暖复古风
```
配色:
- 背景: #f4e4c1 (米黄)
- 强调: #d4574a (砖红)
- 文字: #2c3e50 (深蓝灰)
- 辅助: #8b7355 (棕色)

特点:
- 纸质质感
- 手绘像素
- 温暖舒适
- 参考: Eastward 游戏
```

### 选项 4: 现代暗色风
```
配色:
- 背景: #0f0f0f (纯黑)
- 强调: #7c3aed (紫罗兰)

特点:
- 现代简洁
- 高对比度
- 专业感
- 参考: Linear, Vercel
```

## 实施步骤

### 第一阶段: 静态网站 (1-2天)
1. 选择技术栈和设计风格
2. 搭建基础页面结构
3. 实现内容展示板块
4. 部署到 Vercel

### 第二阶段: 像素可视化 (2-3天)
1. 设计像素办公室场景
2. 实现 Canvas 动画
3. 状态同步机制
4. 集成到网站

### 第三阶段: 自动化 (1天)
1. GitHub Actions 配置
2. 定时构建部署
3. 数据自动更新
