## 筛选标准

一个项目要入选，需要满足以下条件：

**必须满足**（至少 2 项）：
- [ ] 与 AI 内容创作直接相关
- [ ] 有实际演示/Demo 可体验
- [ ] 近期有活跃更新（30天内）
- [ ] 中文文档或社区讨论较多
- [ ] 可以解决当前课程制作中的痛点

**加分项**:
- 有可视化界面（非纯代码工具）
- 支持中文或国内可访问
- 有教程/案例展示
- 社区活跃度高（Issues/PR 活跃）

**排除项**:
- 纯后端/基础设施项目
- 需要复杂部署的企业级工具
- 过于小众没有文档的项目
- 与内容创作无关的工具

## 信息源

1. **GitHub Trending** - 按语言/时间筛选
2. **GitHub Search** - 按关键词搜索，按最近更新排序
