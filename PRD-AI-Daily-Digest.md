# AI Daily Digest - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
AI Daily Digest - 全球人工智能行业每日精选

### 1.2 产品定位
一个专注于 AI 行业的每日新闻聚合网站，为从业者、研究者和爱好者提供精选的全球 AI 要闻、前沿研究、行业观察和数据洞察。

### 1.3 目标用户
- AI 行业从业者（工程师、产品经理、创业者）
- 学术研究人员
- 科技媒体编辑
- AI 爱好者和投资者

---

## 2. 功能需求

### 2.1 核心模块

| 模块 | 说明 | 优先级 |
|------|------|--------|
| 头条新闻 | 当日最重要的 3-5 条 AI 行业新闻 | P0 |
| 前沿研究 | 最新发表的论文、研究成果速览 | P0 |
| 行业观察 | 深度分析、评论、趋势解读 | P1 |
| 今日数据 | 融资、股价、市场数据等 | P1 |
| 本周回顾 | 过去一周的热点汇总 | P2 |

### 2.2 详细功能

#### 2.2.1 头条新闻
- 展示当日精选的 3-5 条重要新闻
- 每条包含：标题、摘要（100字内）、来源、发布时间、原文链接
- 支持标签筛选（如：大模型、自动驾驶、AI 芯片等）
- 支持按时间筛选（今日/昨日/近7天）

#### 2.2.2 前沿研究
- 展示最新 AI 论文（arXiv、顶级会议）
- 每条包含：论文标题、作者、机构、摘要、PDF链接、代码链接
- 支持按领域筛选（CV/NLP/RL/多模态等）
- 支持热度排序（引用数、GitHub stars）

#### 2.2.3 行业观察
- 深度文章、行业报告、专家观点
- 支持长文阅读模式
- 支持收藏功能

#### 2.2.4 今日数据
- 融资动态（公司、金额、轮次、投资方）
- 股市数据（AI 相关上市公司股价）
- 产品发布（新模型、新功能上线）

#### 2.2.5 本周回顾
- 周日自动生成
- 汇总本周各模块热门内容
- 支持邮件订阅推送

## 4. 技术架构

### 4.1 前端
- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **组件库**: shadcn/ui
- **部署**: Vercel / Cloudflare Pages

### 4.2 后端/数据源
- **内容采集**: Python 爬虫 + RSS 订阅
- **数据源**:
  - 新闻：TechCrunch、The Verge、机器之心、量子位
  - 论文：arXiv API、Papers With Code
  - 融资：Crunchbase、IT桔子
  - 数据：Yahoo Finance、CoinMarketCap
- **存储**: Markdown 文件 / Supabase / Notion API
- **自动化**: GitHub Actions 定时抓取

### 4.3 部署流程
```
定时任务 (GitHub Actions)
    ↓
抓取各数据源
    ↓
AI 总结生成摘要
    ↓
写入 Markdown/数据库
    ↓
触发 Next.js 构建
    ↓
部署到 Vercel
```

## 6. 数据流程

### 6.1 内容采集
| 数据源 | 采集方式 | 频率 |
|--------|----------|------|
| arXiv | API | 每小时 |
| TechCrunch | RSS | 每小时 |
| 机器之心 | RSS | 每小时 |
| GitHub Trending | 爬虫 | 每天 |
| Twitter/X | API | 每小时 |

### 6.2 内容处理
1. **抓取**: 获取原始内容
2. **去重**: 基于 URL/标题去重
3. **分类**: AI 自动打标签
4. **摘要**: AI 生成中文摘要
5. **审核**: 人工抽检（可选）
6. **发布**: 写入数据库/文件

## 9. 风险与对策

| 风险 | 对策 |
|------|------|
| 数据源封禁 | 使用 RSS/API，控制请求频率 |
| 内容版权问题 | 只展示摘要，链接到原文 |
| 内容质量不稳定 | AI + 人工抽检 |
| 服务器成本 | 静态生成 + CDN |
